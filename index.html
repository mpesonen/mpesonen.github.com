SniffleCut

Demo GitHub

AI-powered tool that detects and automatically removes sniffle sounds from Final Cut Pro video projects. Uses the BEATS model and a neural network custom-trained on the FluSense dataset to identify sniffles with 98.5% accuracy, then modifies FCPXML project files to remove those segments while maintaining timeline continuity.

  • Real-time sniffle detection with confidence scores
  • 50ms temporal resolution audio analysis
  • Automatic video segment removal and timeline adjustment
  • Retrainable classifier

arXiv Semantic Search

AWS Live Demo GitHub

A semantic search application for arXiv research papers. Search for papers by meaning rather than keywords using vector embeddings. Natural language queries are converted to embeddings and compared against a database of paper embeddings to find conceptually related research.

  • Semantic search powered by sentence-transformers (all-MiniLM-L6-v2)
  • ChromaDB vector database with 384-dimensional embeddings
  • FastAPI REST API with React/TypeScript frontend
  • Deployed on AWS EC2 with Docker

LLM Eval System

AWS Live Demo GitHub

A lightweight system to run, compare, and visualize evaluation suites for Large Language Models. Define test cases in YAML, run them against different models, and track pass rates over time with a web dashboard that highlights regressions and improvements.

Clarified my thinking about LLM system maintainability and scalability (versioned prompts & models, LLM-as-judge testing).

  • YAML-based evaluation suites with expected conditions (contains, JSON validation, length)
  • Pass rate visualization with revision-based tracking
  • Regression detection with color-coded markers
  • Versioned system prompts for A/B testing
